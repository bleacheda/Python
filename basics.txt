3.4.5. DATA STRUCTURES
lists           - A collection which is ordered and changeable. Allows duplicate members. [1, 2, 3]
tuples          - A collection which is ordered and unchangeable. Allows duplicate members. ('Apples', 'Oranges', 'Grapes')
sets            - A collection which is unordered and unindexed. No duplicate members. {4, 5, 6}

dictionaries    - A collection (of {'key': 'value'} pairs) which is unordered, changeable and indexed. No duplicate members.
